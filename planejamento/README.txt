Sistema CRUD de Planejamento do PPA (Órgãos, Programas, Ações, Metas, Iniciativas e Dotações)

Arquivos principais:
- conexao.php
- layout.php
- index.php
- orgaos.php
- programas.php
- acoes.php
- metas.php
- iniciativas.php

Passos básicos:
1. Crie um banco de dados MySQL (por exemplo: 'planejamento').
2. Importe o arquivo planejamento.sql (estrutura das tabelas).
3. Ajuste usuario/senha/host em conexao.php, se necessário.
4. Coloque todos os arquivos em uma pasta no seu servidor PHP (XAMPP, WAMP etc.).
5. Acesse index.php pelo navegador.
